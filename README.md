## Microservicio Estadistica
Este microservicio se encarga de registrar todas las estadisticas de cada jugador de un torneo en concreto, tiene conexion con Auditoria, Partida y usuario y en base a la estadistica se puede sacar el ranking global, puede hacer cambios como guardar, generar , actualizar o eliminar y se guardara en la base de datos para tener un historial de dichos cambios

# Dependencias

* Spring Web
* Validation
* Spring Data JPA
* OpenFeign
* MySQL Driver
* Lombok
